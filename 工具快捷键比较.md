# 查找文件
idea：双击shift；
vscode：ctrl+p 或者 ctrl+e；
notepad++：

# 定位到某行
idea：ctrl+g；
vscode：ctrl+g；
notepad++：ctrl+g；

# 单行注释
idea：ctrl+/；
vscode：ctrl+/；
notepad++：ctrl+k 或者 ctrl+q；

# 多行注释
idea：ctrl+shift+/；
vscode：alt+shift+a；
notepad++：ctrl+shift+q；

# 匹配大括号
idea：
vscode：
notepad++：ctrl+b；

# 下一行
idea：shift+enter；
vscode：ctrl+enter；
notepad++：

# 在文件中查找内容
idea：ctrl+f    ctrl+shift+f；
vscode：ctrl+f    ctrl+shift+f；
notepad++：ctrl+f   ctrl+shift+f；

# 删除行
idea：ctrl+x;
vscode：ctrl+x;
notepad++：ctrl+l；
